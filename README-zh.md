<h1 align="center">战斗优化</h1>

本模组旨在通过提升玩家的机动性来改善星露谷战斗体验

## ❤️ 主要功能

- 在使用武器或工具时允许角色移动（可分别为武器和工具启用/禁用）。
- 在攻击时让角色面向鼠标方向。
- 支持武器自动挥动（如启用，会在按住左键/使用键期间持续挥动）。

## 💻 安装

1. 将下载的 zip 解压到 Stardew Valley 的 Mods 目录。
2. 启动游戏，确保 SMAPI 已加载 Mod。

## ⚙️ 配置
配置项位于 `AttackWhenMoving/config.json`，建议搭配 Generic Mod Config Menu (GMCM) (通用模组配置菜单) 模组修改，支持以下选项：

- `FaceMouseWhenAttack` (bool) - 在攻击时让角色面向鼠标，默认 true。
- `WeaponAutoswing` (bool) - 启用武器自动挥动，默认 true。
- `EnableForWeapons` (bool) - 在武器使用时启用移动，默认 true。
- `EnableForTools` (bool) - 在工具使用时启用移动，默认 true。
