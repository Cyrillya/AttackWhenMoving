[center][size=5][font=Verdana][b]Combat QoL[/b][/font][/size]
[/center]
[center]A mod that improves combat experience in Stardew Valley by increasing player mobility while using weapons or tools
[/center]
[center]Inspired by [i]Terraria[/i][/center]
[center][/center]
[center][/center]
[size=4][font=Verdana][b]❤️ Key Features[/b][/font][/size]
[list]
[*]Allow movement while using weapons or tools (can be enabled/disabled separately for weapons and tools)[/*]
[*]Face the mouse cursor while attacking (also supports gamepad right-stick aiming)[/*]
[*]Weapon autoswing support (when enabled, weapons will continue swinging while the use key is held)[/*]
[*][b]Controller right-stick support[/b] — three modes: Mouse Aim, Directional Attack, Edge Trigger[/*]
[*][b]Cancel special attacks[/b] — special attack animations can be interrupted by normal attacks[/*]
[/list]
[size=4][font=Verdana][b]💻 Installation[/b][/font][/size]
[list=1]
[*]Extract the downloaded zip into the Stardew Valley [i]Mods[/i] folder[/*]
[*]Start the game and ensure SMAPI loads the mod[/*]
[/list]
[size=4][font=Verdana][b]⚙️ Configuration[/b][/font][/size]
The config is located at [i]AttackWhenMoving/config.json[/i]. It is recommended to use [i]Generic Mod Config Menu (GMCM)[/i] for in-game configuration.

[b]Common Settings[/b]
[list]
[*][b]FaceMouseWhenAttack[/b] (bool, default: true) — Face the mouse or right-stick direction when attacking[/*]
[*][b]WeaponAutoswing[/b] (bool, default: true) — Enable weapon autoswing (hold use key to keep swinging)[/*]
[*][b]EnableForWeapons[/b] (bool, default: true) — Allow movement while using weapons[/*]
[*][b]EnableForTools[/b] (bool, default: true) — Allow movement while using tools[/*]
[*][b]SpecialAttackCancellable[/b] (bool, default: true) — Special attack animations can be canceled by normal attacks[/*]
[/list]

[b]Controller Settings[/b]
[list]
[*][b]ControllerAttackMode[/b] (string, default: edge) — Right stick control logic: [i]mouse[/i] (simulate mouse), [i]direction[/i] (independent aim, press use to attack), [i]edge[/i] (auto-attack at stick edge)[/*]
[*][b]ControllerAimDeadZone[/b] (float, default: 0.0) — Right-stick dead zone for facing direction (prevents drift from constant re-aiming)[/*]
[*][b]ControllerAttackDeadZone[/b] (float, default: 0.9) — Right-stick threshold for triggering edge-trigger attack[/*]
[/list]
